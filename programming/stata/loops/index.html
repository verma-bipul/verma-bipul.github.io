For loop in stata

Looping through parallel lists

local agrp "cat dog cow pig"

local bgrp "meow woof moo oinkoink"

local n : word count `agrp'

forvalues i = 1/`n' {
    local a : word `i' of `agrp'
  local b : word `i' of `bgrp'
   di "`a' says `b'"
}

/// Output
cat says meow
dog says woof
cow says moo
pig says oinkoink
