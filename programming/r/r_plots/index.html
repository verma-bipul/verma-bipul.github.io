R Plots

We’ll start by describing the functioning of the inbuilt R plot function, though there are better packages like ggplot.

plot(x_axis_data, y_axis_data, type="l", col="b",
     xlab="label_x", ylab="label_y", 
     xlim=c(0, 10), ylim=c(-10, 20))
title(main="TITLE", sub="subtile")

In addition, labels can be added to a scatter plot in R using the following commands.

plot(x_axis_data, y_axis_data)
text(x_axis_data, y_axis_data, labels= names_vec)

In order to fit a line through the scatter plot we can use abline commands, which adds lines to the existing plots

plot(x_axis_data, y_axis_data)
abline(a, b)  # here a is the intercept while b is the slope for equation of line y ~ a + bx
abline(lm(y_axis_data ~ x_axis_data)) # this fits a linear regression line through the scatter-plot.

gglots

ggplot is a versatile tool. I prefer to use ggplot alongwith dplry, which helps with the clarity of code. Given below is the general structure of ggplot that I follow. I like to divide the plot into three parts - (i) data points and plot attribute, (ii) axis scaling, (iii) labels

Scatterplots

df %>% ggplot(aes(x = col1, y = col2)) +   # this feeds the data
geom_point() +  # to add scatter plots
geom_smooth(method="lm") +  # add linear fit through scatter plot
geom_line() +  # connects the dots with a line
theme_bw() +  # nice simple b/w theme

scale_x_continuous(n.breaks=10) + # number of breaks on x-axis 
scale_y_continuous(n.breaks=10) + 

labs(title="Plot",y="y-lab", x="x-lab") + # to assign labels to axis
geom_text(label=aes(col_with_labels), nudge_x=0.2, nudge_y=0.2 ,check_overlap=T)  # geom_text adds text to the points, nudge adjusts the labels

ggsave("path/fine_name.png")  # save the current plot

ggsave attributes

ggsave("path/filename.png",
       scale = 1,
       width = 2.0, height = 1.5,
        dpi = 300,  # dpi stands for dots per inch 
       limitsize = TRUE,
       units = c("cm"))   # unit size can be changed to in, mm as well
