Apply_fns

There are different kinds of apply functions:

  • apply
  • lapply
  • sapply
  • mapply
  • vapply
  • rapply
  • tappy

The reason that there are so many kinds of apply function in R is that we want to avoid making use of for loop as much as possible. The class of apply functions are more optimized that loops.

apply

apply(df, 1, fun) # Row-wise operation
apply(df, 2, fun) # Column-wise operation
