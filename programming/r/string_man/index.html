String_man

String Manipulation in R

Paste Function

The paste function in R is used to join strings together.

>paste0("a", 1:3)  
> "a1" "a2" "a3"

>paste("a", 1:3)
> "a 1" "a 2" "a 3"

Sprintf Function

The sprintf function prints strings in a specified format.
Examples:

sprintf("%06d", 12)   # "000012" - 6 digits with leading zeros
sprintf("%.2f", 15.45678) # "15.49" - prints upto two decimal places
